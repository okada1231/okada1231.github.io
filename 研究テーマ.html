研究テーマ

複数人の音声認識を行った場合の字幕評価



研究テーマの理由

3年次の臨地実務実習で音声認識を用いて問題解決を検討していた。

実装の難しさとメインの機能として絶対に必要ではないことから実装しなかったが、音声認識について興味を持つきっかけになった。



研究目的

リアルタイムで複数人の音声認識で字幕表示を図った際に字幕精度を落とさずに何人ほどの声を識別できるか。そして、一人の場合と比べて、どれくらい字幕精度を維持できるかを知りたい。



使用環境

mac

Visual Studio Code


プログラミング言語

Python3.10

音声認識モデル

Google Cloud Speech-to-text

Speech Recognition

Whisper
